# Security Policy

## Reporting a Vulnerability

Mapbox takes the security of our software seriously. If you believe you have found a security vulnerability, please report it responsibly through one of the following channels:

- [Mapbox Responsible Disclosure Program](https://www.mapbox.com/platform/disclosure)
- [Mapbox on HackerOne](https://hackerone.com/mapbox?type=team)

Please do not report security vulnerabilities through public GitHub issues.
